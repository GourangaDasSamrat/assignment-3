## How to start development

1. In project root directory, type

```bash
npm install
npm run watch:css
```

2. Then open the `index.html` file in your browser.

## Build process

In project root directory, type

```bash
npm install
npm run build:css
```

### Requirements

- Node 12.13.0 or Higher
- Modern browser ( like Chrome or Firefox)
